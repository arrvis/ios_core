# Getting Started
  
## Requirements

- XCode 10.2~
- Homebrew 2.1.0~
- Carthage 0.32.0~
- Swiftlint 0.27.0~

## Setup

```sh
brew install swiftlint
brew install carthge
brew tap blender/tap https://github.com/blender/homebrew-tap.git
brew install blender/homebrew-tap/rome
bin/bootstrap
```

## Lint

```sh
swiftlint
```
or
```sh
swiftlint autocorrect
```
